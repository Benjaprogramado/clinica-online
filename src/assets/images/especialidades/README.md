# Imágenes de Especialidades

Esta carpeta contiene las imágenes por defecto para las especialidades.

Las imágenes deben estar en formato PNG y tener los siguientes nombres:

- `cardiologia.png`
- `dermatologia.png`
- `gastroenterologia.png`
- `neurologia.png`
- `oftalmologia.png`
- `pediatria.png`
- `psiquiatria.png`
- `traumatologia.png`
- `default.png` (imagen por defecto cuando no se encuentra una especialidad)

Si alguna imagen no existe, se utilizará `default.png` automáticamente.

Las imágenes recomendadas son de 400x200px para el formato rectangular de los botones de especialidades.
